### Nima baholanadi?
- Hisoblab chiqishlar natijasi to‘g‘ri

- O‘zgaruvchilar va funksiyalar faqat a, b, c, d bo‘lib qolmagan, ma’noli nomlarga ega (bu haqda batafsil 2.3-videoda).
